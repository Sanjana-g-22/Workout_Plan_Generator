# Workout_Plan_Generator
Simple workout plan generator build using python only
 Workout Plan Generator

A Python-based workout plan generator with login & registration, built using Tkinter for GUI and MySQL for storing workout data.

Features

User login & registration (CSV-based)

Beginner, Intermediate, and Intense workout plans

Focus area selection (Upper, Lower, Full body)

Workout duration options (15 / 30 / 60 minutes)

Random workout generation per day

Tech Stack

Python

Tkinter

MySQL

CSV
